The Great escape

The Great Escape is een project uit het eerste jaar, van de opleiding ICT-Gamedevelopment op de HVA. De game is gemaakt in Java. Het doel van de game is om de lava zo lang mogelijk voor te blijven, terwijl je over verschillende hindernissen springt.

  • programmeertaal : Java
  • Datum: December 2015
  • OntwikkelOmgeving: Processing

Boss state Diagram

Hieronder is mijn Boss state Diagram. Deze bestaat uit drie state’s met ieder eigen gedrag voor de boss. Verbeteringen in mijn ontwerp

Gisteren heb ik mijn idee gepitcht, en feedback ontvangen van mijn mede studenten. Naar aanleiding hiervan heb ik een lijstje met aanpassingen in mijn concept.

  • In mijn eerste ontwerp wilde ik maar twee kleuren gebruiken, het probleem wat je krijgt is dat het niet duidelijk is met welke objecten in de game wereld interactie mogelijk is. de aanpassing is om de kleur wit toe te voegen voor alle interactieve items
  • Een nieuw idee in het ontwerp is om aan het begin van het level uit te zoemen. Zo ziet de speler pas het level als het spel begint.
  • Over de economy in mijn game heb ik ook veel gesproken met mijn klasgenoten, al is hier nog niet veel concreets uit gekomen. De komende week ga ik beslissen hoe ik de economy in mijn game wil aanpakken.
